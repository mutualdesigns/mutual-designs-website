Requires at least: 2.5

Stable tag: trunk



Allows you to set login redirection based upon user role.





== Description ==



This plugin allows you to redirect users on login based upon their role. It even includes custom roles if you have any.





== Installation ==



1. Upload the plugin to your 'wp-content/plugins' directory

1. Activate the plugin
